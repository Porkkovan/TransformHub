Pre-Demo Checklist

✅ Services Running
□
Next.js frontend
http://localhost:3000
□
FastAPI backend
http://localhost:8000/api/v1/health
□
PostgreSQL
Postgres.app running · transformhub DB
□
Demo data seeded (3 orgs)
npx tsx prisma/seed.ts
🖥️ Browser Setup
□ Open incognito/private window to avoid cached session issues
□ Zoom browser to 90% so all columns are visible
□ Pre-login as demo@transformhub.ai and navigate to Dashboard
□ Select US Bank org — confirm KPIs are loaded (9 products visible)
□ Have this demo guide open in a separate tab
⚡ Start Commands
# Terminal 1 — Backend
cd TransformHub/agent-service
source venv/bin/activate
uvicorn app.main:app --reload --port 8000

# Terminal 2 — Frontend
cd TransformHub/nextjs-app
npm run dev
🆘 Troubleshooting
Wrong org loads
localStorage.removeItem("currentOrgId")
Port 3000 taken
Kill Product Coach: kill $(lsof -ti:3000)
No data showing
npx tsx prisma/seed.ts
